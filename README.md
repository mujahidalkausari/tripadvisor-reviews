# tripadvisor-reviews
Extract 5 latest reviews from tripadvisor.com first page.

Tools: BeautifulSoup and Requests
Note: The code is set according to the requirements of AWS Lambda function. It can be uploaded directly to AWS with virtualenv site-packages as a zip file

The same repository has tripadvisor.py which is build using scrapy which handles pagination and scrape all other pages.
